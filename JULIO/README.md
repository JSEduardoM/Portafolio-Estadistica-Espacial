# 🌌 The Spatial Nexus - Portafolio de Estadística Espacial

Bienvenido a **The Spatial Nexus**, el portafolio académico y profesional de **Julio Segundo Eduardo Maquera**. Este proyecto integra análisis avanzados de datos geográficos, modelado estadístico y visualización interactiva para transformar datos complejos en conocimiento estratégico.

![Preview](https://raw.githubusercontent.com/JSEduardoM/Portafolio-Estadistica-Espacial/main/tareas/tarea-02-analisis-espacial/figuras/image.png)

## 🚀 Proyectos Destacados

### 1. 📊 Análisis de Pérdidas Agrícolas (ENA 2014-2024)
Un sistema de información geográfica (SIG) interactivo desarrollado con Leaflet.js para analizar el impacto de fenómenos climáticos en el agro peruano.
- **Tecnologías:** Leaflet.js, Chart.js, GeoJSON.
- **Funcionalidades:** Línea de tiempo interactiva, mapas de calor, filtrado por variables de riesgo (Heladas, Plagas, Sequía).

### 2. 🐄 Dashboard Ganadero
Herramienta de análisis para el tratamiento de datos de hatos de vacas, optimizando el seguimiento de producción y salud animal.
- **Tecnologías:** R Shiny, ETL, Data Cleaning.

### 3. 📄 Monografía: Introducción a la Estadística Espacial
Documentación técnica sobre los fundamentos de los datos espaciales y sistemas de coordenadas.

### 4. 📄 Solución de Ejercicios: Distribución Gaussiana
Desarrollo analítico de ejercicios prácticos sobre la distribución Gaussiana (normal) multivariante, funciones de densidad de probabilidad y sus aplicaciones geoestadísticas.

## 🛠️ Stack Tecnológico
- **Frontend:** HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6+).
